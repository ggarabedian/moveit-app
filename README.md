## Installing and running the project

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### `npm install`

After cloning the repository locally, run the `npm install` command to install the app dependencies.

### `npm start`

After installing the dependencies, run the `npm start` command to start the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view the app in the browser.

The page will reload if you make changes to the code.\
